POST /api/transcribe

Direct REST endpoint to transcribe short video/audio clips with Sarvam STT cloud and style them with Fireworks GLM-5.

Form Data Parameters

  • file: Video or audio bytes (MP4, WEBM, WAV, MP3). Max 500MB.
  • style: Caption personality vibe (formal, sarcastic, humorous_tech, humorous_non_tech).

cURL Request Example

curl -X POST https://api.vdcap.ai/v1/transcribe \
  -H "api-subscription-key: vdcap_live_19f2yvfb_nrrXb..." \
  -F "file=@my_clip.mp4" \
  -F "style=sarcastic"

Success Response (200 OK)
